# Download JSON file in posh themes directory.
## Open terminal and type:
```bash
nano ~/.config/fish/config.fish
```

## Add:
```bash
oh-my-posh init fish --config ~/.config/bombaclat.omp.json | source
```
## Save and type:
```bash
source ~/.config/fish/config.fish
```
